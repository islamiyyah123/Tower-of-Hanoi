# Tower-of-Hanoi
A Python script that solves the classic mathematical puzzle, the Tower of Hanoi. It uses a recursive algorithm to find the optimal moves and prints out the exact state of the rods after every single step.

## 🚀 Features
* **Recursive Logic**: Implements an optimized recursive function to solve the puzzle.
* **State Tracking**: Visually tracks the disks across all three rods as lists in real-time.
* **Clean Strings**: Joins and formats the entire solving history into a readable timeline output.

## 🛠️ How to Run
1. Make sure you have Python installed.
2. Run the file: `python tower_of_hanoi.py`
